# UESTC 电子科技大学网络认证脚本
------------------------------
这次更新主要是修好了「自动连接校园网」脚本在新版登录页面下无法使用的问题。
学校把上网登录页面和后台接口升级之后，原来的脚本拿不到正确的 IP 和登录凭证，就会一直报错登录失败。

本次更新做了几件事：

- 重新适配了新的登录页面格式，能正确识别出当前电脑在校园网内的 IP 地址。

- 模拟浏览器的完整登录流程（先打开登录页，再向后台接口申请登录令牌），而不是直接“硬请求”接口。

- 保留原来的使用方式不变，用户只需要填写学号和密码，脚本就可以再次自动完成拨号登录。

### 搞这干啥？
- 学校的电信宽带自动掉线太频繁了，移动的稍好一点但也不行（尊贵的移动还屏蔽了游戏串流软件，真是谢谢你）
- 校园网很稳定，很少掉线。但是如果再出现封在家里一个月没法回学校的情况，那就得想办法让电脑一直在线了，不然没法给老板打工。
### 怎么用？
- 支持登录以下类型的网络（我都试过的）：
```angular2html
- 校园网有线接入+学号认证（在主楼做的测试，至今可用）
- 移动、电信寝室宽带有线接入+学号认证（类似于硕丰6、7、8组团那种插网线直接弹出认证页面的，至今可用）
```
只需修改config.py就行。然后手动注销断网，输入：
```angular2html
python login_once.py
```
如果能联网，那说明配置没问题。
之后一直运行always_online.py就行了。
```angular2html
python always_online.py
```
